Markdownパーサーを作り始めた

Markdownパーサーを作り始めた。

今まで人生で一度もパーサー的な実装をしたことがなかったことと、このブログの実装において利用しているremarkを自作のもので置き換えてみたい欲から作り始めた。CommonMark 0.29 の仕様にのっとって実装を進めている。headers, code blocksなど各仕様ごとに実例があるのでそれをテストケースとしてテストしている。

各テストケースをpassできるように実装していくため、目に見えて進捗率がわかりやすいのでモチベーションを維持しやすいのがとても良い（といってもまだ5日目）。毎朝少しずつ進めていき、将来的にはブログのmarkdownのパース処理を置き換えたい。
